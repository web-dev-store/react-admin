## Usage

```shell
git clone https://github.com/web-dev-store/react-admin.git

cd react-admin

npm install

npm run start

# open browser and visit http://localhost:3000
```

## Features

- TypeScript
- Login
- Custom layout
- Route and auth
- Widget dashboard
- Grid demo
- Form demo
- Mock server
- Git hook
